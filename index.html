Sobre Mí & Habilidades

Soy estudiante de Ingeniería en Sistemas Computacionales en el Instituto Tecnológico de Oaxaca (ITO). Me apasiona resolver problemas complejos a través de la programación y crear soluciones de software eficientes tanto en entornos locales como web.

Mis habilidades técnicas incluyen lenguajes como Java, C#, JavaScript, además de desarrollo web moderno utilizando el Stack MERN y librerías como Bootstrap. También tengo un fuerte interés en la administración de redes y la gestión de infraestructura tecnológica.

DonaVida

Plataforma de vinculación logística para la donación de sangre en Oaxaca, desarrollada para el evento INNOVATEC con la categoria CERTAMEN DE PROYECTOS. Este proyecto fue implementado utilizando una arquitectura de microservicios y el Stack MERN (MongoDB, Express, React, Node.js).

Sistema de Lavandería

Aplicación de escritorio desarrollada en C# WPF para la gestión de un negocio local. Incluye módulos funcionales para el cálculo de peso, manejo de inventario y generación/impresión de tickets para los clientes.

Sistema de inventario

Plataforma web diseñada para el control integral de mantenimiento, gestión de tickets y administración de inventario (dispositivos y consumibles). El frontend de la aplicación fue construido con HTML, CSS y JavaScript, mientras que la lógica y almacenamiento del backend están impulsados por Node.js y MongoDB, gestionando el entorno con dependencias de npm.
